About the Project

This project converts Indian Sign Language gestures into text using computer vision and machine learning. The system detects hand landmarks from live camera input and classifies gestures to generate text output with predictive suggestions.

Key Features

Real-Time Detection

Captures live gestures using webcam input.

Hand Landmark Extraction

Uses MediaPipe to detect 21 hand keypoints.

Gesture Classification

Machine learning model predicts gesture characters.

Text Prediction

Autocomplete suggestions improve typing efficiency.

Technology Stack

  • Python
  • OpenCV
  • MediaPipe
  • TensorFlow / Keras
  • Streamlit
